Java Collections In Depth

by Alonso del Arte

Nonfiction > Computer programming > Java

Paperback, $38.99

The Java Development Kit (JDK) offers a lot more options for gathering objects into collections than just the famous Array­List<E> from the java.util package, ranging from extending the provided abstract classes and implementing the provided interfaces to simply using the available concrete classes.

In this book, Al explains how to leverage the Java collections class hierarchy and ex­ten­si­bil­i­ty with concepts gleaned from Scala's immutable col­lec­tions to create customized solutions for the project at hand. Includes exercises and "recipes," plus a link to a companion website with all the source.

Critics hail Alonso del Arte's début novel as "Wait, this is not a novel. Why was I assigned to review this? The author explains things very clearly, but he also assumes knowledge that the intended audience for this book has that I don't have. Jimmy Olsen's gonna get fired. Perry! Perry!?" (Cat Grant, Daily Planet).
