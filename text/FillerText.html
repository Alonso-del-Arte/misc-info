Filler Text

Filler text. This is filler text. It does not communicate anything par­tic­u­lar­ly im­por­tant, but it should at least be gram­mat­i­cal­ly cor­rect. I believe that filler text should be in the same language as the larger text where it is to function as a placeholder.

Generally, filler text should not contain statements that are provably true or provably false. That can be accomplished through vagueness, by withholding the necessary context to ascertain the veracity of a given statement.

Or, alternatively, filler text can contain statements that are provably true, but trivially so. Such as, for example, that true is the opposite of false and false is the opposite of true.
